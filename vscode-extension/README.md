# Emacs Themes for VS Code

This extension brings popular Emacs themes to Visual Studio Code. The themes have been carefully converted from their original Emacs versions to provide a similar look and feel in VS Code.

## Available Themes

### Spacemacs
The iconic Spacemacs themes collection, known for its elegant design and carefully chosen colors:
- **Spacemacs Dark** - The default dark variant, optimized for long coding sessions
- **Spacemacs Light** - A clean and readable light variant
- **Spacemacs** - The original base theme

### Leuven
A highly legible light theme originally designed for use with Org mode in Emacs.

## Installation

1. Open VS Code
2. Press `Ctrl+P` to open the Quick Open dialog
3. Type `ext install emacs-themes` to find the extension
4. Click Install
5. Click Reload to reload VS Code

## Selecting a Theme

1. Open the Command Palette (`Ctrl+Shift+P`)
2. Type "Color Theme" and select "Preferences: Color Theme"
3. Choose one of the Emacs themes from the list

## Theme Sources
